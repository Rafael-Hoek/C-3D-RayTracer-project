Instruction manual for the use of Rafael Hoek's mesh optimizer:

- run "py pipeline.py" in a shell within this folder
- enter the relative name of the unoptimized mesh file, including extension
- enter the name of the desired output mesh file, including extension
- enter the maximum number of triangles that may be present in any box.
	- Hint: a good value for a mesh of approximately 1 million triangles is 40
